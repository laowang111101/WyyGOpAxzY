# 学生成绩管理系统 python1054

## 项目概述

学生成绩管理系统是一个基于Python技术和Django框架开发的B/S模式应用程序。它使用MySQL数据库进行数据存储，主要功能是管理学生和教师的成绩信息，以及与之相关的各项功能。

## 技术栈

- 后端：Python
- Web框架：Django
- 数据库：MySQL

## 功能模块

系统主要包括以下功能模块：

- **首页**：展示系统概览和通知公告。
- **个人中心**：用户可以管理个人信息。
- **教师管理**：对教师信息进行增删改查。
- **学生管理**：对学生信息进行增删改查。
- **公告信息管理**：发布、编辑和删除公告信息。
- **课程类型管理**：对课程类别进行管理。
- **课程信息管理**：对课程详细信息进行管理。
- **选课信息管理**：管理学生选课信息。
- **成绩信息管理**：管理学生成绩信息。

## 系统角色

系统主要定义以下角色：

- 管理员：负责系统整体管理和维护。
- 教师：可以查看和管理自己教授的课程及学生成绩。
- 学生：可以查看和选课，查看个人成绩。

## 运行环境

- Python环境（推荐使用3.x版本）
- Django框架
- MySQL数据库

## 部署步骤

1. 配置Python环境。
2. 安装Django框架和MySQL数据库。
3. 创建数据库实例和用户权限。
4. 导入数据库结构。
5. 配置系统设置和静态资源。
6. 运行Django开发服务器。
